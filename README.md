# Minecraft clone using
c++, openGL 3+, sfml, glm, parallel_hash_map, FastNoise

# Controls
| Key | Description |
| ------ | ------ |
| w,a,s,d | movement |
| right shift | flight on/off |
| left shift | down when flight on |
| space | jump/up |
| f3 | hide/disply debug info |

# Features
- infinite world (from zero to billions)
- "alive" water
- shadows
